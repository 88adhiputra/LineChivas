# LineChivas
_May the Chivas be with you..._

Tutorial
------
Recomended for python 2.7

Using git :

    $ git clone https://github.com/merkremont/LineChivas


Python packages :

    $ pip install thrift==0.9.3
    $ pip install rsa
    $ pip install requests

Run Chivas :

    $ python chivasbot.py
